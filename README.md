## Getting started

```bash
git clone https://github.com/luisMakesIt/admin-usuarios-spring-sbadmin-jwt-restfull-.git
cd admin-usuarios-spring-sbadmin-jwt-restfull-
```

### Install dependencies

```bash
./mvnw test  # or mvn test when no wrapper exists
```

### Run locally

```bash
./mvnw spring-boot:run
```
